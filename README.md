# Smart Route Management System (SRMSS)

## Setup Instructions

### Prerequisites
- Node.js v18+
- MySQL 8.0
- MySQL Workbench

### Installation
1. Clone repository
2. Run `npm install`
3. Create database `srmss_db` in MySQL
4. Import `srmss_backup.sql` 
5. Update `db.js` with your credentials
6. Run `node app.js`

### Login Credentials
- Admin: admin@srmss.com
- Depot Officer: depot@srmss.com / password
- Supervisor: supervisor@srmss.com / password
